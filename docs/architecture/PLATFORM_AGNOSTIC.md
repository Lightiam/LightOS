# LightOS Platform-Agnostic Architecture

LightOS is a platform-agnostic OS for AI accelerators, inspired by OpenCL.

## Supported Platforms
- NVIDIA GPUs (CUDA)
- AMD GPUs (ROCm)
- ARM GPUs (OpenCL)
- Intel XPUs (oneAPI)
- Apple Silicon (Metal)
- Future Photonic NPUs

## Benefits
- No vendor lock-in
- 27-30% cost savings
- Cloud-agnostic deployment
- Future-proof
