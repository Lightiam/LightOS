üöÄ Quick Start

Get up and running with LightOS in 5 minutes.

Step 1: Install LightOS

# Clone repository
cd ~
git clone https://lightos.repo/LightOS.git
cd LightOS

# Run installer (5-10 minutes)
sudo ./simple-deploy.sh

# Verify installation
lightos --version

Step 2: Launch Interactive UI

lightos

Interactive Menu Options:

  • Quick Fine-tune with Unsloth (‚ö° 2-5x faster)
  • Start GLM-4 or Qwen2.5-Coder agents
  • Check hardware capabilities
  • View DCIM dashboard

Step 3: Generate Code with AI

# Generate code
lightos-code qwen generate "FastAPI REST API with JWT auth"

# Explain code
lightos-code glm4 explain mycode.py

# Fix bugs
lightos-code qwen fix broken.py

Step 4: Fine-Tune a Model

# Quick fine-tune (30 minutes)
lightos-train train llama-3.1-8b alpaca

# With custom dataset
lightos-train train llama-3.1-8b my_dataset.json --max-steps 200

Common Commands

# Interactive launcher
lightos

# Training
lightos-train
lightos-train list
lightos-train train  

# Coding agents
lightos-code qwen generate "prompt"
lightos-code glm4 explain file.py
lightos-code qwen fix file.py

# Check GPU
nvidia-smi

Performance Tips

For Limited Memory (8GB VRAM)

# Use smaller models
lightos-code qwen generate "code" --model-size 1.5b

# Enable 4-bit quantization
lightos-train train llama-3.1-8b alpaca --4bit

For Maximum Speed

# Use Flash Attention
lightos-train train llama-3.1-8b alpaca --flash-attention

# Reduce sequence length
lightos-train train llama-3.1-8b alpaca --max-seq-length 1024
