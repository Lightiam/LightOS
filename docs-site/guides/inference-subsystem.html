Architecture

Module 1: Platform-Agnostic Abstraction

// Write once, run anywhere
auto device = LightAccelerator::create(DeviceType::NVIDIA_GPU, 0);
auto kernel = device->compile_kernel(source, "matmul");
device->launch_kernel(kernel, config, args);

Module 2: Thermal-Aware Scheduler

  • Predictive Cooling: Pre-cool zones before heavy workloads
  • Sparsity-Aware: 30% power reduction for sparse models
  • Topology-Aware: Migrate jobs to cooler devices

Module 3: Tile-Based Execution

Double Buffering: Compute tile N while transferring N+1

Result: Hide 300ns HBM latency, achieve 92% MFU

Getting Started

git clone https://github.com/Lightiam/LightOS.git
cd LightOS/inference-subsystem
mkdir build && cd build
cmake -DCMAKE_BUILD_TYPE=Release -DENABLE_CUDA=ON ..
make -j$(nproc)
sudo make install
